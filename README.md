# obs-auto-replay-buffer

ゲーム起動時に自動でOBSのリプレイバッファをONにするツールです。

## 概要

- OBSの**ゲームキャプチャ設定を自動で読み取り**、リプレイバッファを自動開始
- ゲーム起動時に**OBSのシーンも自動で切り替え**（Apexを起動したら「Apexシーン」へ）
- ゲーム終了時に自動停止（設定でオフにしてずっとオンのままにすることも可能）
- 複数ゲームの**優先順位を設定可能**（コバークス起動中にApexを開いたらApexへ切り替え）
- タスクバーのトレイに常駐して動作
- OBS WebSocket API を使ってOBSを制御

---

## 使い方（PC初心者向け）

### 1. OBSを設定する

**WebSocket設定を有効にする**

OBS → ツール → WebSocketサーバー設定 → 「WebSocketサーバーを有効にする」にチェック

パスワードをメモしておいてください。

**ゲームキャプチャのモードを設定する（重要）**

ゲームキャプチャソースのプロパティ → モード → **「特定のウィンドウをキャプチャ」** を選択し、ウィンドウにゲームを指定する

> ⚠️ 「特定のウィンドウをキャプチャ」モードにすることで、ゲームの自動認識・シーン自動切替が機能します。

### 2. exeをダウンロードする

[Releases](../../releases) から最新の `obs-auto-replay-buffer.zip` をダウンロードして解凍してください。

### 3. 起動する

`obs-auto-replay-buffer.exe` をダブルクリックします。

初回起動時に設定画面が表示されるので、OBSのパスワードを入力して「設定を保存して始める」をクリックします。

### 4. 完了！

あとはゲームを起動するだけで自動でリプレイバッファがONになり、そのゲームのシーンに切り替わります。

---

## シーンの自動切り替えについて

OBSのシーン構成がこうなっている場合：

```
シーン「Apex」    → ゲームキャプチャ（r5apex_dx12.exe）
シーン「Overwatch」→ ゲームキャプチャ（overwatch.exe）
```

Apexを起動すると自動で「Apex」シーンに切り替わってリプレイバッファが開始されます。
追加設定は不要で、OBSのゲームキャプチャ設定を読み取って自動で判断します。

---

## ゲームの優先順位について

複数のゲームを同時に起動している場合（例：コバークスでウォームアップ中にApexを起動）、どちらを優先するか設定できます。

トレイ → **「ゲーム設定（手動登録）」** を開くと、画面下部に「ゲームの優先順位」セクションが表示されます。

- OBSに登録されているゲームが自動で一覧に表示されます
- **↑↓ボタン**で並び替え → 上にあるゲームが優先されます
- 「保存して閉じる」で設定完了

**例：ApexをKovaaK'sより上に設定した場合**
- KovaaK's起動中にApexを開く → Apexシーンへ自動切り替え ✓
- Apexプレイ中にKovaaK'sが起動 → 無視（Apexのまま） ✓

---

## トレイアイコンのメニュー

タスクバー右下の `∧` をクリックするとアイコンが表示されます。右クリックでメニューが開きます。

| メニュー | 説明 |
|----------|------|
| 監視を開始 / 停止 | ゲーム監視のオン・オフ |
| ゲーム設定（手動登録） | フルスクリーンモード等でOBSから自動取得できない場合の手動追加・優先順位設定 |
| Windows起動時に自動起動 | PC起動時に自動で常駐させる（チェックで切り替え） |
| 終了 | アプリを終了 |

---

## フルスクリーンモードを使っている場合

OBSのゲームキャプチャを「任意の全画面アプリケーション」モードで使っている場合は、exe名を自動取得できません。

その場合はトレイ → **「ゲーム設定（手動登録）」** からゲームのexe名を手動で追加してください（1回だけ）。

手動追加したゲームも優先順位リストに表示されます。

---

## config.jsonの設定（上級者向け）

```json
{
  "obs_host": "localhost",
  "obs_port": 4455,
  "obs_password": "パスワード",
  "poll_interval": 5,
  "stop_on_game_exit": true,
  "games": [],
  "priority_order": ["r5apex_dx12.exe", "overwatch.exe"]
}
```

| 設定項目 | 説明 |
|----------|------|
| `poll_interval` | ゲーム検出の間隔（秒）（デフォルト: 5） |
| `stop_on_game_exit` | ゲーム終了時にリプレイバッファを止めるか（`false` にするとずっとオン） |
| `games` | 手動登録したゲームのリスト（OBSから自動取得できない場合に使用） |
| `priority_order` | ゲームの優先順位（exe名のリスト、上が高優先度）※設定画面からGUIで設定可能 |
