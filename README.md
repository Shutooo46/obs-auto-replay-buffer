# obs-auto-replay-buffer

ゲーム起動時に自動でOBSのリプレイバッファをONにするPythonスクリプトです。

## 概要

- OBS起動時ではなく、**指定したゲームの起動を検出したとき**にリプレイバッファを自動開始
- ゲームごとに有効/無効を設定可能
- OBS WebSocket API を使ってOBSを制御

## 必要環境

- OBS Studio 28以降（WebSocket機能内蔵）
- Python 3.10以降
- obsws-python
