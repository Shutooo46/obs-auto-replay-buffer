# obs-auto-replay-buffer

ゲーム起動時に自動でOBSのリプレイバッファをONにするツールです。

## 概要

- OBS起動時ではなく、**指定したゲームの起動を検出したとき**にリプレイバッファを自動開始
- ゲーム終了時に自動停止（設定でオフにしてずっとオンのままにすることも可能）
- GUIでゲームを簡単に追加・管理可能
- タスクバーのトレイに常駐して動作
- OBS WebSocket API を使ってOBSを制御

---

## 使い方（PC初心者向け）

### 1. OBSのWebSocket設定を有効にする

OBS → ツール → WebSocketサーバー設定 → 「WebSocketサーバーを有効にする」にチェック

パスワードを設定した場合はメモしておいてください。

### 2. exeをダウンロードする

[Releases](../../releases) から最新の `obs-auto-replay-buffer.zip` をダウンロードして解凍してください。

### 3. 起動する

`obs-auto-replay-buffer.exe` をダブルクリックします。

初回起動時に設定画面が表示されるので、OBSのパスワードを入力して「設定を保存して始める」をクリックします。

### 4. ゲームを登録する

タスクバー右下のアイコンを右クリック → **「ゲーム設定」**

1. 登録したいゲームを起動する
2. 「起動中から選ぶ」ボタンでゲームを選択
3. ゲーム名を入力して「追加」
4. 「保存して閉じる」→ 監視を再起動

これでゲームを起動するたびに自動でリプレイバッファがONになります！

---

## トレイアイコンのメニュー

| メニュー | 説明 |
|----------|------|
| 監視を開始 / 停止 | ゲーム監視のオン・オフ |
| ゲーム設定 | ゲームの追加・削除・有効無効の管理 |
| Windows起動時に自動起動 | PC起動時に自動で常駐させる（チェックで切り替え） |
| 終了 | アプリを終了 |

---

## config.jsonの設定（上級者向け）

```json
{
  "obs_host": "localhost",
  "obs_port": 4455,
  "obs_password": "パスワード",
  "poll_interval": 5,
  "stop_on_game_exit": true,
  "games": [
    { "name": "Apex Legends", "exe": "r5apex.exe", "enabled": true }
  ]
}
```

| 設定項目 | 説明 |
|----------|------|
| `poll_interval` | ゲーム検出の間隔（秒）（デフォルト: 5） |
| `stop_on_game_exit` | ゲーム終了時にリプレイバッファを止めるか（`false` にするとずっとオン） |

## 開発者向け：ビルド方法

```bash
pip install -r requirements.txt
build.bat をダブルクリック
```

`dist\obs-auto-replay-buffer\` フォルダが生成されます。
